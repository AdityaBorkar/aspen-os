| `source_module` | text | Source module name |
| `source_entity_id` | text | Source entity ID (nullable) |
| `source_entity_type` | text | Source entity type (nullable) |
| `branch` | text (FK → branch) | Branch (nullable) |
| `connection` | text (FK → connection) | Connection (nullable) |
| `obligation_id` | text (FK → compliance_obligation) | Originating obligation (nullable) |
| `created_by` | text | Creator user ID |
| `assigned_to` | text | Assigned user (nullable) |
| `assigned_reviewer` | text | Assigned reviewer (nullable) |
| `attachment` | text | Storage key (nullable) |
| `issue_date` | date | Issue date (nullable) |
| `effective_date` | date | Effective date (nullable) |

| `metadata` | jsonb | Flexible metadata |

| `created_at` | timestamptz | Creation timestamp |
| `updated_at` | timestamptz | Last update timestamp |

## Relationships

```text
compliance_obligation ──1:N──→ compliance_document (obligation_id)
compliance_document   ──self-ref──→ compliance_document (renewed_from)
branch                ──1:N──→ compliance_document (branch)
connection            ──1:N──→ compliance_document (connection)
compliance_document   ──1:N──→ compliance_audit_entry (entity_id)
compliance_obligation ──1:N──→ compliance_audit_entry (entity_id)
```
