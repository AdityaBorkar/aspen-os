## Event Map

### Document Lifecycle Events

| Event | Payload | Trigger |
|---|---|---|
| `compliance:document_created` | `{ document: { category, id, name } }` | Document created |
| `compliance:document_updated` | `{ document: { id, name }, changes }` | Document updated |
| `compliance:document_submitted` | `{ documentId, submittedBy }` | Document submitted for review |
| `compliance:document_verified` | `{ category, documentId, sourceEntityId, sourceModule, verifiedBy }` | Document verified |
| `compliance:document_rejected` | `{ category, documentId, reason, rejectedBy, sourceEntityId, sourceModule }` | Document rejected |
| `compliance:document_completed` | `{ completedAt, documentId, referenceNumber, sourceEntityId, sourceModule }` | Document completed |
| `compliance:document_archived` | `{ documentId }` | Document archived |
| `compliance:document_renewed` | `{ newDocumentId, oldDocumentId }` | Document renewed |
| `compliance:document_reviewer_assigned` | `{ documentId, reviewerId }` | Reviewer assigned |
| `compliance:document_attachment_uploaded` | `{ documentId, storageKey }` | Attachment uploaded |
| `compliance:document_snoozed` | `{ documentId, snoozedBy, snoozedUntil }` | Document snoozed |
| `compliance:document_generated` | `{ documentId, obligationId, sourceModule }` | Auto-generated from obligation |

### Expiry & Reminder Events

| Event | Payload | Trigger |
|---|---|---|
| `compliance:document_expiring` | `{ daysUntilExpiry, documentId, sourceEntityId, sourceModule }` | Expiry reminder threshold reached |
| `compliance:document_due` | `{ daysUntilDue, documentId, sourceEntityId, sourceModule }` | Due date reminder threshold reached |
| `compliance:document_expired` | `{ category, documentId, sourceEntityId, sourceModule }` | Document expired |
| `compliance:document_overdue` | `{ category, daysOverdue, documentId, sourceEntityId, sourceModule }` | Document overdue |
| `compliance:document_escalated` | `{ daysSinceExpiry, documentId, escalationLevel }` | Escalation triggered |

### Obligation Events

| Event | Payload | Trigger |
|---|---|---|
| `compliance:obligation_created` | `{ obligation: { category, id, name } }` | Obligation created |
| `compliance:obligation_activated` | `{ obligationId }` | Obligation activated |
| `compliance:obligation_deactivated` | `{ obligationId }` | Obligation deactivated |
| `compliance:obligation_updated` | `{ obligation: { id, name }, changes }` | Obligation updated |

### System Events

| Event | Payload | Trigger |
|---|---|---|
| `compliance:weekly_summary` | `{ summary: { total, verified, expired, expiringSoon, overdue, activeObligations, documentsGenerated } }` | Weekly summary report |
| `compliance:scheduled_job_executed` | `{ jobName, executionTime, recordsProcessed, errors }` | Scheduled job completed |

## Usage

### Subscribing to Events

```ts
// React to document verification
await framework.pubsub.subscribe(
  "compliance:document_verified",
  async (message) => {
    const { documentId, verifiedBy, category } = message.data;
    // Send notification, update dashboard, etc.
  },
);

// React to expiry reminders
await framework.pubsub.subscribe(
  "compliance:document_expiring",
  async (message) => {
    const { documentId, daysUntilExpiry } = message.data;
    // Send email notification
  },
);

// Listen to weekly summary
await framework.pubsub.subscribe(
  "compliance:weekly_summary",
  async (message) => {
    const { summary } = message.data;
    // Push to dashboard, send report email
  },
);
```
