The Organization module defines 7 tables and 5 enums using Drizzle ORM.

## Enums

| Enum | Table Column | Values |
|---|---|---|
| `organization_status` | `organization.status` | `active`, `suspended`, `archived` |
| `branch_type` | `branch.type` | `headquarters`, `office`, `warehouse`, `store`, `factory`, `remote`, `other` |
| `connection_type` | `connection.type` | `client`, `vendor`, `partner`, `subsidiary`, `parent_company`, `investor`, `regulator`, `insurer`, `bank`, `other` |
| `connection_status` | `connection.status` | `active`, `inactive`, `prospect`, `former` |
| `connection_note_type` | `connection_note.type` | `general`, `call`, `email`, `meeting`, `contract_renewal`, `issue` |

## Tables

### `organization`

The root aggregate — represents a company/organization.

| Column | Type | Description |
|---|---|---|
| `id` | text (PK) | UUID |
| `name` | text | Organization name |
| `slug` | text (unique) | URL-safe identifier |
| `status` | organization_status | `active` / `suspended` / `archived` |
| `email` | text | Contact email |
| `phone` | text | Contact phone |
| `website` | text | Website URL |
| `logo` | text | Storage key for logo |
| `accent_color` | text | Hex color (default `#3B82F6`) |
| `locale` | text | Locale (default `en-US`) |
| `timezone` | text | Timezone (default `UTC`) |
| `metadata` | jsonb | Flexible metadata |
| `created_at` | timestamptz | Creation timestamp |
| `updated_at` | timestamptz | Last update timestamp |

| Column | Type | Description |

| `type` | branch_type | Branch type |

| `founded_date` | date | Founded date |
| `opened_date` | date | Opened date |
| `metadata` | jsonb | Flexible metadata |
| `created_at` | timestamptz | Creation timestamp |

## Relationships

```text
organization ──1:N──→ branch (hierarchical, self-ref parent_branch)
organization ──1:N──→ connection
connection   ──1:N──→ connection_contact (cascade delete)
connection   ──1:N──→ connection_note (cascade delete)
```

Addresses and bank accounts are standalone tables (reusable across entities).
