The Drive module defines 8 tables and 4 enums using Drizzle ORM.

## Enums

| Enum | Values |
|---|---|
| `drive_item_type` | `file`, `folder` |
| `drive_grantee_type` | `user`, `group` |
| `drive_permission` | `viewer`, `editor`, `owner` |
| `drive_public_link_permission` | `view`, `edit` |

## Tables

### `drive_folder`

| Column | Type | Description |
|---|---|---|
| `id` | text (PK) | `gen_random_uuid()::text` |
| `name` | text | Folder name |
| `owner_id` | text | Owner user ID |
| `parent_id` | text | Parent folder (nullable, FK → `drive_folder.id`) |
| `path` | text | Full path (unique), e.g. `/Documents/Reports` |
| `color` | text | Hex color label (nullable) |
| `description` | text | Description (nullable) |
| `is_trashed` | boolean | Soft-delete flag (default `false`) |
| `trashed_at` | timestamptz | When trashed (nullable) |
| `created_at` | timestamptz | Creation timestamp (default `now()`) |
| `updated_at` | timestamptz | Last update timestamp (default `now()`) |

### `drive_file`

| Column | Type | Description |
|---|---|---|
| `id` | text (PK) | `gen_random_uuid()::text` |
| `name` | text | File name |
| `owner_id` | text | Owner user ID |
| `folder_id` | text | Parent folder (nullable, FK → `drive_folder.id`) |
| `path` | text | Full path (unique), e.g. `/Documents/Reports/report.pdf` |
| `content_type` | text | MIME type |
| `size` | bigint | File size in bytes |
| `storage_key` | text | Object key in storage backend |
| `etag` | text | ETag from storage (nullable) |
| `description` | text | Description (nullable) |
| `version` | integer | Current version number (default `1`) |
| `is_trashed` | boolean | Soft-delete flag (default `false`) |
| `trashed_at` | timestamptz | When trashed (nullable) |
| `created_at` | timestamptz | Creation timestamp (default `now()`) |
| `updated_at` | timestamptz | Last update timestamp (default `now()`) |

### `drive_file_version`

| Column | Type | Description |
|---|---|---|

### `drive_label`

| Column | Type | Description |
|---|---|---|
| `id` | text (PK) | `gen_random_uuid()::text` |
| `name` | text | Label name |
| `color` | text | Hex color |
| `is_global` | boolean | Org-wide vs personal (default `false`) |
| `owner_id` | text | Owner for personal labels (nullable) |

### `drive_item_label`

| Column | Type | Description |
|---|---|---|
| `id` | text (PK) | `gen_random_uuid()::text` |
| `item_id` | text | File or folder ID |
| `item_type` | drive_item_type | `file` or `folder` |
| `label_id` | text | Label (FK → `drive_label.id`) |
| `applied_by` | text | User who applied the label |
| `applied_at` | timestamptz | When applied (default `now()`) |

- Unique index on `(item_id, item_type, label_id)` prevents duplicate label assignments.

### `drive_public_link`

| Column | Type | Description |

## Relationships

```text
drive_folder ──self-ref──→ drive_folder (parent_id)
drive_folder ──1:N──→ drive_file (folder_id)
drive_file   ──1:N──→ drive_file_version (file_id)
drive_label ──1:N──→ drive_item_label (label_id)
drive_folder/file ──1:N──→ drive_item_label (item_id + item_type)
drive_folder/file ──1:N──→ drive_share (item_id + item_type)
drive_folder/file ──1:N──→ drive_public_link (item_id + item_type)
drive_public_link ──1:N──→ drive_access_log (public_link_id)
drive_folder/file ──1:N──→ drive_access_log (item_id + item_type)
```
